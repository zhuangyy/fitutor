# 健身好搭子 — 用户手册

## 一、软件概述

**软件名称：** 健身好搭子

**版本号：** v1.2.3

**开发语言：** Dart（Flutter 框架）

**运行平台：** Android、iOS、macOS

**软件类型：** 健身工具类应用

**软件简介：**
"健身好搭子"是一款 100% 离线运行的健身动作编排和跟练应用。使用者可自由编排训练计划，自定义每个动作的组数、时长和休息时间；训练过程中 App 通过语音播报动作名称和倒计时，就像一位私人教练在耳边指导。不需要注册账号，所有数据存储在本地，不上传至任何服务器。

---

## 二、主要功能

### 2.1 训练计划管理

- **新建计划：** 点击首页右下角「新建计划」按钮，输入计划名称和描述
- **编辑计划：** 长按已有计划卡片进入编辑模式
- **拖拽排序：** 拖动计划卡片右侧手柄可调整计划顺序
- **删除计划：** 在计划编辑页中可删除计划

### 2.2 动作编排

- **添加动作：** 在计划编辑页点击「添加动作」，从动作库中选择
- **动作库：** 预置 7 个基础动作（杠铃卧推、杠铃深蹲、硬拉、哑铃弯举、杠铃肩推、卷腹、开合跳），覆盖胸/腿/背/手臂/肩/腹/全身七大肌群
- **自定义动作：** 可创建自己的训练动作，填写名称、目标肌群和类型（力量/计时）
- **参数设置：** 每个动作可独立设置组数、训练时长、次数、组间休息和动后休息

### 2.3 训练跟练

- **开始训练：** 点击计划卡片，确认后进入训练模式
- **语音播报：** 训练过程中语音播报当前动作名称、剩余秒数（可关闭）
- **倒计时圆环：** 红色（训练中）/ 橙色（组间休息）/ 绿色（动后休息）三色区分不同阶段
- **暂停/继续：** 训练或休息时点击屏幕任意位置暂停（可在设置中关闭）
- **跳过休息：** 休息阶段可点击「跳过休息」提前进入下一组
- **训练进度：** 底部显示进度条和动作列表，已完成 ✓、进行中 ●、待完成 ○
- **结束训练：** 点击左上角关闭按钮，确认后结束并保存训练记录

### 2.4 训练记录

- **历史列表：** 查看所有已完成和未完成的训练记录
- **记录详情：** 点击记录可查看每个动作的完成情况（绿色完成 / 橙色未完成）
- **删除记录：** 在详情页可删除不需要的记录

### 2.5 设置

- **语音播报：** 开关 TTS 语音播报
- **震动反馈：** 开关训练阶段切换时的震动提示
- **间隔提醒：** 设置在倒计时中每隔几秒播报剩余时间（关闭/5秒/10秒/15秒/20秒/30秒）
- **快捷暂停：** 开关点击屏幕任意位置暂停功能

## 三、技术架构

### 3.1 整体架构

```
UI (pages/widgets) → State (Provider) → Services → Data (sqflite DAOs)
```

### 3.2 核心模块

| 模块 | 功能 |
|---|---|
| `models/` | 数据模型：Exercise（动作）、TrainingPlan（训练计划）、PlanExercise（计划-动作关联）、WorkoutSession（训练记录）|
| `database/` | SQLite 数据库层，含 DatabaseHelper 和三个 DAO（ExerciseDao、PlanDao、SessionDao）|
| `services/` | 核心服务：CoachEngine（训练状态机）、TtsService（语音播报）、HapticService（震动）、BackgroundServiceManager（后台保活）|
| `providers/` | 状态管理：PlanProvider、WorkoutProvider、SettingsProvider（基于 ChangeNotifier）|
| `pages/` | 7 个页面：首页、计划编辑、动作管理、训练页、训练总结、历史、设置 |
| `widgets/` | 可复用组件：CountdownRing（倒计时圆环）、PlanCard（计划卡片）|

### 3.3 训练状态机

```
IDLE → WORKING → RESTING → WORKING → ... → POST_EXERCISE_RESTING → TRANSITIONING → ...
                                                                                      ↓
                                                                                 COMPLETED
```

- **WORKING:** 训练计时中，使用动作的 workSeconds
- **RESTING:** 组间休息，使用 restSeconds
- **POST_EXERCISE_RESTING:** 动作完成后休息，使用 afterRestSeconds
- **PAUSED:** 暂停状态，可随时从 WORKING/RESTING/POST_EXERCISE_RESTING 进入
- **COMPLETED:** 所有动作完成

### 3.4 数据存储

SQLite 数据库（v4），包含 4 张表：

| 表名 | 说明 | 关键字段 |
|---|---|---|
| exercises | 动作库 | name, category, muscle_group, is_preset |
| training_plans | 训练计划 | name, description, sort_order |
| plan_exercises | 计划-动作关联 | plan_id, exercise_id, sets, work_seconds, rest_seconds |
| workout_sessions | 训练记录 | plan_name, duration_sec, exercises_json, finished_at |

## 四、安装与运行

### 4.1 系统要求

- Android 5.0 及以上
- 运行内存 ≥ 2GB

### 4.2 安装说明

1. 从应用商店下载 APK 安装包
2. 允许安装未知来源应用（如非应用商店下载）
3. 首次启动无需注册，直接使用

### 4.3 权限说明

| 权限 | 用途 |
|---|---|
| 语音播报（TTS） | 训练过程中播报动作和倒计时 |
| 震动 | 训练阶段切换时提供触觉反馈 |
| 通知 | 后台训练时保持播报 |

本应用不访问网络、不读取通讯录、不获取位置信息。
